Weight and speed

Weight is a property present in all robots. It governs how fast the robots move and how efficient they're able to climb slopes. Usually, the bigger and more sensors any given robot has, the heavier it will be, forcing users to make a compromise between accuracy (of their readings) and speed. The weight values of each robot are present in their respective pages.

Bot controller

Bot controller (or bc) is the class that controls the virtual robot inside the simulator. Commands usually use the bc class instance.

Example: bc.wait(1000);

IO

Below are listed three ways information can be passed or shown to/by the robot.

Console

The console can be found on the sidebar menu, and all values printed to it will be shown there.

LED

The LED is a RGB light located at the top of the robot. The user can light one of the basic color values ("WHITE", "BLACK", "YELLOW", "MAGENTA", "GREEN", "BLUE", "CYAN", "RED") or a customizable RGB value from 0 to 255.

Sensors

Sensors are the only input components for the bot controller, and are further described here.
